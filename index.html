Misskey Auth

FlutterアプリケーションでMisskeyのOAuth認証とMiAuth認証を簡単に実装できるライブラリです。

🚀 主な機能

  • OAuth 2.0認証: Misskey 2023.9.0以降のサーバーでOAuth 2.0認証をサポート
  • MiAuth認証: 古いバージョンのMisskeyサーバーでMiAuth認証をサポート
  • 自動認証方式選択: サーバーの対応状況に応じて自動的に適切な認証方式を選択
  • セキュアなトークン保存: flutter_secure_storageを使用してトークンを安全に保存
  • PKCE対応: OAuth 2.0でPKCE（Proof Key for Code Exchange）をサポート
  • 埋め込みWebView不使用: 外部ブラウザを使用してセキュリティを確保
⚠️ 重要
このページはMisskey OAuth 2.0認証のclient_idとして使用されています。 認証時にこのページの情報が表示され、ユーザーがアプリの詳細を確認できます。

📦 インストール

dependencies: misskey_auth: ^1.0.0

🔧 基本的な使用方法

import 'package:misskey_auth/misskey_auth.dart'; final client = MisskeyAuthClient(); final config = AuthConfig( host: 'misskey.io', clientId: 'https://yourusername.github.io/misskey_auth', // このページのURL clientSecret: 'your_client_secret', redirectUri: 'your_app://oauth/callback', appName: 'Your App Name', ); final result = await client.authenticate(config);

🔗 リダイレクトURI設定

このページには以下のリンクタグが含まれています：

<link rel="redirect_uri" href="misskey_auth://oauth/callback">

アプリケーションでは、このURIスキーム（misskey_auth://）でリダイレクトを受け取るように設定してください。

🛡️ セキュリティ

このライブラリは以下のセキュリティ機能を提供します：

  • PKCE（Proof Key for Code Exchange）による認証コード傍受攻撃の防止
  • flutter_secure_storageによるトークンの安全な保存
  • 外部ブラウザを使用した認証（WebView不使用）
  • サーバー情報の事前検証
